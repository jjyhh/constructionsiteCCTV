## 프로젝트 소개
이 프로젝트는 <strong>건설 현장의 안전성을 향상</strong>시키기 위한 것입니다.

### 프로젝트 목표
- 안전하지 않은 상황을 감지하고 경고하는 AI 시스템 구축
- CCTV 기술과 딥러닝 모델을 융합하여 정확한 위험 판단

### 주요 기능
1. 실시간 영상 분석
2. 위험 상황 경고 및 알림
3. 통계 및 보고서 생성
